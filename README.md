# Plant Health: Automated Disease Detection using Transfer Learning

## Project Overview
This project implements a deep learning system for automated crop disease identification from leaf images. By leveraging **MobileNetV2** and **Transfer Learning**, the model accurately detects various plant pathologies to support global food security.

## Technical Specifications
* **Dataset:** PlantVillage (Tomato & Potato Leaves)
* **Architecture:** MobileNetV2 (Pre-trained on ImageNet)
* **Input Size:** 224x224 px
* **Framework:** TensorFlow / Keras

## Performance Results
* **Training Accuracy:** ~98.4%
* **Validation Accuracy:** ~96.2%
* **Inference Speed:** Real-time (<10ms)

## Conclusion
The implementation confirms that Transfer Learning is highly effective for specialized botanical datasets, providing an ideal balance between computational efficiency and diagnostic accuracy.
